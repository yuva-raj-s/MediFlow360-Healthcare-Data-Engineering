# Incident Response Plan
1. **Declare**: Acknowledge alert in Teams. Type "Looking into it".
2. **Triage**: Determine impact. (Is dashboard down? Is data wrong?)
3. **Mitigate**: Apply temporary fix (e.g., restart SHIR).
4. **Resolve**: Deploy code fix.
5. **Post-Mortem**: Document in `15_incidents_and_struggles/`. Share with CIO.
