Deployment Sequence

> FOLLOW PROTOCOL TO REPLICATE MEDIFLOW360 ENVIRONMENT.

PHASE 01

Foundation & Networking

Establish the secure backbone. Provision the Resource Group, Virtual Network (VNet), and Subnets. Configure Private Endpoints for all storage and database services to ensure zero public exposure.

PHASE 02

Governance & Storage

Setup ADLS Gen2 with Hierarchical Namespace. Initialize the Unity Catalog Metastore and assign it to the workspace. Create catalogs and schemas for the Medallion layers.

./create_dirs.ps1
PHASE 03

Security & Connectivity

Configure Azure Key Vault and create Secret Scopes. Deploy the Self-Hosted Integration Runtime (SHIR) for secure on-prem access to the Chennai HIS MySQL database.

PHASE 04

Operational Metadata

Provision Azure SQL DB for operational telemetry. Execute DDLs to initialize the Watermark control system and Audit logging tables.

PHASE 05

Ingestion Orchestration

Deploy the 7 ADF pipelines. Implement the Master Orchestrator with dynamic metadata mapping and delta-load logic for all heterogeneous sources.

PHASE 06

Medallion Processing

Implement the Databricks notebook suite. Transition from DBFS mounts to Service Principal authentication. Deploy SCD Type 2 logic for history tracking in Silver Delta tables.

PHASE 07

Enterprise Serving

Provision Azure Synapse Dedicated SQL Pool. Configure PolyBase loading patterns to move Gold aggregates into Hash-distributed analytics tables.

PHASE 08

Observability & Alerts

Setup Logic Apps as a centralized alert router. Connect ADF and Databricks failure triggers to the Logic App for real-time Teams/Email notifications.
